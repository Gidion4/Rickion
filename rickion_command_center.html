SECURITY

All keys live in localStorage on this machine only. Never transmitted to anyone except their respective APIs. Rickion does not phone home.

  • Gemini · used as primary engine for chat + analysis
  • Claude · reserve, only for hardest prompts (auto-route or /deep)
  • MEXC · spot trading + portfolio reads (no withdraw)
  • Etherscan · on-chain ETH transaction reads
  • Phantom · Solana wallet (Phantom holds key, Rickion only reads address)
HOW TO ROTATE A KEY
  1. Click EDIT next to any service in the table above
  2. Goes to Configuration view, key field pre-focused
  3. Paste new key, click SAVE
  4. Click TEST here in API Vault to verify the new one works

⚠ Best practice: rotate API keys every 30–90 days. Revoke leaked keys immediately at the provider's dashboard.
